Hooks

Hooks are basically functions that get called when events happen in-game, like chat messages.

Functions

Hook.Add(eventname, hookname, func)

 View source »

Adds a hook.

Parameters

  • eventname string

    event name

  • hookname string

    hook name

  • func function

    callback

Example Usage

Hook.Add("characterDeath", "characterDeathExample", function(character)
   print(character)
end)

Hook.Call(eventname, parameters)

 View source »

Calls a hook.

Parameters

  • eventname string

    event name

  • parameters table

    parameters to be passed in

Example Usage

Hook.Add("think", "happyDebuggingSuckers", function()
     Hook.Call("characterDead", {}) -- ruin someone's day
end)

Hook.Remove(eventname, hookname)

 View source »

Removes a hook.

Parameters

  • eventname string

    event name

  • hookname string

    hook name

Example Usage

Hook.Remove("characterDeath", "characterDeathExample")
